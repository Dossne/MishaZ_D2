# Pacman40000

## Версия Unity

- `6000.3.11f1`

## Основные правила игры

- Игрок управляет Пакмэном в лабиринте.
- Цель: собрать все обычные точки (конфеты) и большие бонусные точки (тортики).
- За обычную точку начисляется `10` очков.
- За большую бонусную точку начисляется `50` очков и включается усиление примерно на `8` секунд.
- Во время усиления можно съедать привидений (по `200` очков за каждое).
- Если столкнуться с привидением без усиления, теряется жизнь.
- Стартовое количество жизней: `3`.
- Победа: собраны все точки.
- Поражение: закончились жизни.

## Управление

- Движение: `Arrow Keys` или `WASD`.
- Перезапуск после победы/поражения: `R`.
